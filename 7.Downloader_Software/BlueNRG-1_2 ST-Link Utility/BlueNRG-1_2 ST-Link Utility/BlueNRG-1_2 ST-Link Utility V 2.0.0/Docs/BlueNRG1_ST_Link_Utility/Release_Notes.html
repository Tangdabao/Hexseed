BlueNRG-1 ST-Link Utility (STSW-BNRG1STLINK)
Release Notes

This document contains the release notes of the BlueNRG-1 ST-Link Utility SW package (STSW-BNRG1STLINK) supporting BlueNRG-1, BlueNRG-2 devices.

  • BlueNRG-1 ST-Link Utility SW package includes the BlueNRG-1 ST-Link Utility and CLI PC applications for Windows PC.
  • This document describes the new features that are introduced in each new release of the SW package.
  • A PC running Windows Vista/7 operating systems (64 and 32 bit version) or Windows XP operating systems (32 bit edition), with available USB ports is required for installing the BlueNRG-1 ST-Link Utility SW package.
